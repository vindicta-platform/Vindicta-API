# Vindicta-API Constitution

**Version**: 1.0.0 | **Ratified**: 2026-02-01

## Core Principles

### I. Async-First
All endpoints MUST be async. No blocking calls.

### II. OpenAPI Compliance
All endpoints MUST be fully documented with OpenAPI schemas.

### III. Dependency Injection
Services MUST be injected, not instantiated inline.

### IV. Error Handling
All errors MUST return structured JSON with actionable messages.
